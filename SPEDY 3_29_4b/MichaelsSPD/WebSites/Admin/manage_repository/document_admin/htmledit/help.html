QWebEditor User Manual

Toolbar


New Clear current editor content.
Save Submit current editor content.
Copy Copy current selected text to clipboard.
Cut Remove current selected text and move it to clipboard.
Paste Insert the text from the clipboard to the selected location.
Paste Plain Text Remove formatting from the text in the clipboard and insert the text to the selected location.
Paste From Word Remove Word specific formatting from the text in the clipboard and insert the text to the selected location.
-- Paragraph -- Choose a paragraph format.
-- Size -- Choose a font size.
-- Font -- Choose a font face
-- Style -- Choose a font style.
Foreground Color Apply color to the current selection
Highlight Color Highlight the current selection with a specific color.
Foreground Color Apply color to the current selection
Bold Make the selection bold.
Italic Make the selection italic.
Underline Formats the current selection with a continous underline.
Strikethru Cross out the text in the current selection.
Superscript Formats the current selection as superscript.
Subscript Formats the current selection as superscript.
Decrease Font Size Decrease Font Size.
Increase Font Size Increase Font Size.
Outdent Outdent the current paragraph (move left).
Indent Indent the current paragraph (move right).
Bulleted List Apply or remove bulleted list.
Number List Apply or remove number list.
Align Left Align current paragraph to the left edge.
Align Center Align center.
Align Right Align current paragraph to the right edge.
Align Justify Align current paragraph to the left and right edge.
Hyperlink Insert new link to current selection or edit current link's properties.
Horizontal Rule Insert horizontal rule.
Table Insert a new table or edit current table's properties.
Table Border Hide or show table border guideline.
Image Insert a new image or edit current image's properties.
Insert Symbol Insert Symbol.
HTML Source Show or hide HTML source of current content.

Context Menu



Remove Format Remove current selection format (span tag and class attributes may be left untouched.)
Remove All Format Remove formatting of whole document (span tag and class attributes will be all cleared.)
Remove Link Remove hyperlink from current selection.
Hyperlink Insert new link to current selection or edit current link's properties.
Insert column before Insert a column before the current column of a table.
Insert column after Insert a column after the current column of a table.
Insert row above Insert a row above the current row of a table.
Insert row below Insert a row above the current row of a table.
Delete column Delete current column of a table.
Delete row Delete current row of a table.
Delete table Delete current selected table.
Table properties Edit the current selected table's properties.
Cell properties Edit the current selected cell/column/row properties.
Merge with right cell Merge the current table cell to the one right to it.
Merge with bottom cell Merge the current table cell to the one under it.
